# Spotify UI Clone

A static frontend copy of Spotify's design using only HTML and CSS.

## About
This is a frontend-only project that looks like Spotify. It doesn't play real music - it's just the visual design.

## Features
- Looks like Spotify's interface
- Works on desktop and mobile
- Smooth hover effects
- Clean design

## Technologies Used
- HTML
- CSS

## How to View
1. Download the project
2. Open `index.html` in your web browser

## Note
This is a learning project. I don't own Spotify's design - just recreated it for practice.
